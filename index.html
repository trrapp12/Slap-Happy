THE GAME

Think of it as an onerous, high-stakes game of hangman. The game uses Math.random() to choose a word from pre-defined dictionary. A system of functions
iterate through the word,
count the number
of characters,
and display an equivalent
number of "_" 's to show
you how many letters there are.


Then you type your guess
and an onclick event
activates more funcions
which compare the
characters in the guess array
to the selected word, decrements
the amount of tries, and
determines if you have lost.


Be ready to not just be beat
mentally, but also psychologically.
Yes, we will mock you into
submission. Are you ready?


GO!

Rules:

Player starts with 10 tries to guess a letter in the word. Each wrong guess counts. Correct guesses do not. If you fail to guess the word in 10 guesses YOU WILL BE SHAMED!!!
